# Models License Information

This directory contains TensorFlow Lite (.tflite) models used for age estimation and gender classification within the Android application.

## Included Models

### Age & Gender Estimation Models
- **Files**: `model_age_*.tflite`, `model_gender_*.tflite` (High accuracy)
- **Files**: `model_lite_age_*.tflite`, `model_lite_gender_*.tflite` (Lightweight/High speed)

These models are optimized for mobile inference and are typically trained on demographic datasets such as IMDB-WIKI or UTKFace.
